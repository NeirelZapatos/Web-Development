There are three types of positioning

  • Static

    • Just stacks the elements

  • Relative

    • takes static and can move it based on that static position

  • Absolute

    • takes the closest ancestor (that has position property set) and position based on that. If N/A then the top left of the webpage

  • Fixed

    • Relative to the browser even when you are scrolling
