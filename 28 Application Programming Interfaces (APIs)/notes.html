rest API is most important

endpoint: https://bored-api.appbrewery.com/random

query parameters: https://bored-api.appbrewery.com/endpoint?query=value&query2=value. used for filtering

Path parameters: https://bored-api.appbrewery.com/endpoint/{path-parameter}. used for identifying

API authetication

  • 0. No authetication
  • 1. Basic authentication (uses a password and username in BASE64)
  • 2. API Key Authorization (client that is allowed is use a server typically with an api key)
  • 3. Token Based Authentication (username and password and api token. They are tied together to authorize)
